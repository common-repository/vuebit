Enables gif-like previews for every youtube video on your site.

== Description ==

Vuebit automatically creates gif-like previews for every youtube video on your site. It also supports vimeo, dailymotion and streamable embedded videos. This plugin loads vuebit.js in your pages.

== Installation ==

1. Install the plugin through the WordPress plugins screen directly or upload the plugin files to the `/wp-content/plugins/vuebit` directory.
1. Activate the plugin through the 'Plugins' screen in WordPress.
1. No more actions are required. The script will be added in every page of your blog.

== Frequently Asked Questions ==

= Do I need a subscription with vuebit.com ? =

Yes. In order to use this plugin you need to have an active subscription to one of vuebit's monthly plans. 
Visit https://vuebit.com for more details.

= Does vuebit support only youtube videos ? =
No. Vuebit can create previews for vimeo, dailymotion and streamable as well.
